# Substack Monitoring Dashboard V5

This version uses the Substack archive API first, then RSS fallbacks. It also includes a verified fallback for The Jason Jones Show so the dashboard will not drop Jason Jones entirely if Substack returns a temporary feed/API issue.

## Render settings

- Build command: `npm install`
- Start command: `npm start`
- Runtime: Node
- Root directory: leave blank

## Sources

- The Jason Jones Show
- Persecuted Church Alerts

Alexandrya Pouliot is handled as an author/writer filter, not as a direct feed.
